Percentages based on confidence when programming, as well as how often the languages are used in the total number of projects. Based on the following formula, these percentages are calculated when given a new program: (Quantity of Projects/ Total Number of Projects) * (Scale of Comfort / 10) (on a scale of 1-10, how comfortable are you when you need to code in this language) * 100. For example, 90% of my programs are in C/C++, I am fairly confident I can code in this language so I would give it a comfort scale of 10. Therefore, 0.9 * 1 * 100 = 90%.
